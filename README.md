# LOL-Data-View
beta1.0

英雄联盟各大职业比赛爬虫，并且做数据分析和可视化

本项目通过对

http://lpl.qq.com/

http://www.famulei.com/

https://www.lolesports.com (需要翻墙)

利用Scrapy框架进行数据爬取，利用JsonPath对json数据进行解析，最后将数据利用matplotlib通过图表形式展现
